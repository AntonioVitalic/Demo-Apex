# Demo-Apex -  Ejercicio Técnico Fullstack - Facturas

Repo para implementar el ejercicio técnico (backend + frontend).

## Stack (objetivo)
- Backend: ASP.NET Core 8 (API REST) + SQLite (EF Core)
- Frontend: React + Vite (TypeScript)

## Cómo ejecutar (placeholder)
> Estos pasos se completarán a medida que se agreguen los proyectos.

### Backend
```bash
cd backend/src/InvoiceManager.Api
dotnet restore
dotnet run
```


### Frontend
```bash
cd frontend
npm install
npm run dev
```
